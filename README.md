# Tutorail 
基於Flask 框架設計簡易網站，並串接line robot 

1.至 https://developers.line.biz/zh-hant/ 開通帳號


2.至 heroku.com 使網站上線


3.填寫自己的  your_user_id , channel access token , channel Secret
